## 15장 let, const

- let 은 **선언 단계**와 **초기화 단계**가 분리되어 진행
  - 일시적 사각지대(TDZ; Temporal Dead Zone) 발생
- let 키워드로 선언한 전역 변수는 전역 객체 프로퍼티 아님
  - let 전역변수는 실행 컨텍스트 내에 존재
